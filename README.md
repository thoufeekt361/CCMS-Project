# Campus Complaint Management System (CCMS)

## Overview
The Campus Complaint Management System (CCMS) is a web application that allows students to submit and track campus-related complaints. Administrators can manage complaints, update their status, and monitor complaint statistics through a dashboard.

## Features
- Student Registration and Login
- JWT Authentication
- Submit Complaints
- Upload Complaint Images
- View and Track Complaint Status
- Search and Filter Complaints
- Admin Dashboard
- Manage Students
- Manage Complaints
- Dashboard Charts and Statistics
- Forgot Password via Email

## Technology Stack

### Frontend
- React
- TypeScript
- Tailwind CSS
- Axios
- Chart.js

### Backend
- Node.js
- Express.js
- MongoDB
- JWT Authentication
- Bcrypt

## Installation

1. Clone the repository
   ```bash
   git clone https://github.com/thoufeekt361/CCMS-Project.git
   ```

2. Install dependencies

   Frontend:
   ```bash
   cd frontend
   npm install
   npm run dev
   ```

   Backend:
   ```bash
   cd backend
   npm install
   npm start
   ```
